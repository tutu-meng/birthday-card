# 爱的印象派·人生画展 | Birthday Card

A beautiful digital birthday card featuring Monet-inspired artwork with LINE characters (Brown & Cony).

## Project Overview

This is a 3D folding birthday card that will be hosted on Vercel. The card features:
- A museum exhibition VIP invitation-style front cover
- Impressionist-style artwork pages inside
- A heartfelt Chinese birthday message
- Romantic beach sunset back cover

## Technology Stack

- **HTML5** - Semantic structure
- **CSS3** - Styling with CSS variables, 3D transforms
- **Vanilla JavaScript** - Interactivity
- **Google Fonts** - Noto Serif SC (Chinese) & Playfair Display (English)

## Quick Start

1. Open `index.html` in a browser, or
2. Deploy to Vercel for production
